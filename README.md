# anon-batch-upload
This is a simple batch script to easily upload files to Anon Files. 
# You Need
A folder called Upload in C: (C:\Upload)
A File To Upload

# Instructions
Run Script
Place File in C:\Upload
Type File Name (Example: mynewsoftware.exe)
Press Enter and Copy Link.
